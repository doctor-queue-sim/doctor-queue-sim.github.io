# Симуляция очереди к врачу

Интерактивная веб-симуляция системы массового обслуживания (СМО) для моделирования очереди пациентов к врачу.

## 🎯 Описание

Это приложение демонстрирует работу системы массового обслуживания с использованием дискретно-событийного моделирования. Симуляция позволяет визуализировать процесс прихода пациентов, их ожидание в очереди и обслуживание врачами.

## ✨ Возможности

### Управляемые параметры:
- **Интенсивность прихода** - количество пациентов в час (Пуассоновское распределение)
- **Среднее время приёма** - продолжительность обслуживания в минутах (экспоненциальное распределение)
- **Количество врачей** - от 1 до 10 врачей
- **Длина очереди** - максимальное количество мест в очереди

### Метрики производительности:
- **Время ожидания** - среднее и 95-й перцентиль
- **Пропускная способность** - количество обслуженных пациентов в час
- **Загрузка врачей** - процент времени занятости
- **Длина очереди** - текущая и максимальная
- **Статистика** - количество обслуженных и отказанных пациентов

### Визуализация:
- Анимированное отображение пациентов в очереди
- Визуализация врачебных кабинетов
- Плавные переходы пациентов между состояниями
- Индикация статуса врачей (свободен/занят)

## 🔬 Математическая модель

### Приход пациентов
Используется **Пуассоновское распределение**:
- Интервалы между приходами следуют экспоненциальному распределению
- Формула: `interArrivalTime = -ln(1 - random()) / λ`
- где λ - интенсивность прихода (пациентов/минуту)

### Время обслуживания
Используется **экспоненциальное распределение**:
- Формула: `serviceTime = -mean * ln(1 - random())`
- где mean - среднее время обслуживания

### Метрики
- **Среднее время ожидания**: `Σ(waitTime) / n`
- **95-й перцентиль**: значение, ниже которого находятся 95% наблюдений
- **Пропускная способность**: `servedPatients / simulationTime`
- **Загрузка врачей**: `(busyTime / totalTime) * 100%`
